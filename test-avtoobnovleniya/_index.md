Для работы с платформой [**Gram.ax**](http://Gram.ax) необходимо понимать, что она состоит из двух основных частей: десктопного приложения для редактирования (работает с Git) и портала документации, который можно развернуть на своем сервере для читателей.



Схема грамакса - приложение для создания контента и редактирования, гит как хранилище, и портал <http://mpknowledge/> для просмотра, также админка для добавления каталогов на портал.

Скрин с надписью где что - где каталог, где пространство
