Портал документации <http://mpknowledge/> представляет из себя сайт, расположенный на корпоративном сервере:

![](./4-sinkhronizaciya-izmeneniy.png){width=1162px height=301px}

:::info 

Доступ к ресурсу возможен только из внутренней сети компании.

:::

Информация, расположенная в каталогах портала автоматически обновляется из центрального хранилища каждые 3 минуты.

Для управления отображаемыми каталогами портала существует режим администратора.

:::info 

Для добавления или удаления каталогов необходимо обращаться к Смакаеву Равилю.

:::
